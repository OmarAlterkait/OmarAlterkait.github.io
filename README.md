# omaralterkait.github.io

Personal website of Omar Alterkait, served by GitHub Pages at
[omaralterkait.github.io](https://omaralterkait.github.io/).

Plain HTML/CSS/JS, no build step.

- `index.html` - single page: about, research, publications, news
- `contact.html` - contact information
- `css/style.css` - site styling (dark theme, CSS variables at the top)
- `js/main.js` - email de-obfuscation and background canvas animation
- `assets/` - CV PDF and images (`assets/img/omar.jpg` is the headshot)
